# 🏏 Virat Kohli ODI Career – EDA Project

This project performs Exploratory Data Analysis (EDA) on Virat Kohli's One Day International (ODI) cricket career using Python.

---

## 📌 Project Overview

The goal of this project is to analyze match-by-match ODI data and discover insights about:

- Total runs and batting average  
- Strike rate analysis  
- 50s and 100s count  
- Year-wise performance  
- Performance against different teams  
- Dismissal patterns  
- Boundary contribution  
- Consistency using rolling average  

---

## 📂 Dataset

- File: `Virat_Kohli_ODI.xlsx`
- Contains match-level statistics such as:
  - Runs
  - Balls Faced
  - Strike Rate
  - Fours and Sixes
  - Opposition
  - Batting Position
  - Dismissal
  - Match Date
